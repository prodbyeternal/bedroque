# bedroque
**bedroque** is a for-fun movement mod for Source Engine games built on the Portal 2 branch. Originally created as an addon for *The Stanley Parable*, it enhances movement mechanics and adds a variety of customization tools for experimentation and practice.

The mod is built on top of the **SourceAutoRun** plugin, commonly used within the speedrunning community.

---

## ✨ Features

### 🕹️ Movement Enhancements

* Bunnyhopping (auto-hop support)
* Duckjump unlock
* Air acceleration overrides
* Built-in movement binds:

  * Long Jump (LJ) *(currently limited on this engine branch, to be worked)*
  * Multi Jump (MJ) *(currently limited on this engine branch, to be worked)*
  * Jump Bug (JB) *(currently limited on this engine branch, to be worked)*
* Advanced techniques:

  * Edgebugging
  * Pixelsurfing*

### 🎨 Visual Customization

* Click-based GUI menu
* Field of View (FOV) changer
* Movement trails
* CS:GO-style crosshair
* Bind indicators
* Velocity graph

---

## ⚠️ Important Notes

* `sv_cheats` is enabled **by default** when using this mod.
* Achievements will be disabled while the mod is active.
* Not recommended for use in official or competitive speedruns.

---

## 📌 Disclaimer

bedroque is intended for **practice, experimentation, and fun only**. Use at your own discretion.
